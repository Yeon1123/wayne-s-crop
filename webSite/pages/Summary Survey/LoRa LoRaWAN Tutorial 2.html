LoRa/LoRaWAN Tutorial 2 : What is LoRa and LoRaWAN

LoRa : A gateway can handle hundreds of devices at the same time. In this project, we use Draginoâ€™s LG01.

Lora end node consists of a radio module with antenna, and a microprocessor to process such as sensor data. End nodes also are often battery powered.

Lora gateway consists of an antenna and microprocessor. The antenna is used to access the internet while the microprocessor is used to process tasks. Multiple gateways can receive data from the same end node. It can get receive multiple frequencies simultaneously.

LoRa does not support direct communication to LoRa devices without the use of gateways. If desired, Radiohead Packet Radio library can be used for an embedded microprocessor to achieve communication directly between the devices.
