LoRa/LoRaWAN Tutorial 3 : Rules and Regulations

  1. ISM Band
    LoRa operates in the unlicensed ISM radio band that are available worldwide. Each reagion have right frequency.

    Microwave ovens, medical equipments or baby monitors are using ISM band.
    • Advantage : Anyone is allowed to use this frequencies, just no license fee!
    • Disadvantage : Low data rate and lots of interference.

  2. Rules.
    There are some organizations which manage the radio spectrum. All countries are using the standard sets by either ETSI and FCC. (Except Japan and South Korea)
    • ETSI : European Telecommunications Standard Institute
    • FCC : US Federal Communications commission

  3. Time on air & Duty cycle
    When receiver gets a signal from sender, it takes a certain amount of time called ToA(Time on Air). For example, if ToA is 400ms and Duty cycle is 1%, that means the next signal will be send after 39.6s. Duty cycle is regulated by standard.
